## Quick start

## 1.install

### Steps

1. Type following commands:
``` console
npm install
or
yarn install
```

2. Serve your app:
``` console
npm start
or
yarn start
```
## 2.Build
```sh
npm run build or yarn build
```

Builds the app for production to the `build` folder.<br>

Your app is ready to be deployed.
